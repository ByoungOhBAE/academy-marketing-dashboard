# 🎯 학원 마케팅 대시보드

부산 한국조리기능장요리발효학원의 블로그 및 인스타그램 콘텐츠 관리 대시보드

## 📋 기능

- 블로그/인스타그램 콘텐츠 미리보기
- A/B/C 버전별 원고 확인
- 작성 완료된 콘텐츠 목록 관리

## 🚀 실행 방법

```bash
npm install
npm start
```

서버가 `http://localhost:3000`에서 실행됩니다.

## 📂 데이터 구조

`public/data.json`에 콘텐츠 데이터가 저장됩니다:
- `blog[]`: 블로그용 콘텐츠
- `insta[]`: 인스타그램용 콘텐츠

## 🔧 기술 스택

- Node.js + Express
- Vanilla JavaScript
- JSON 기반 데이터 관리
